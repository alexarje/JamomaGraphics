Jamoma Graphics
Cross-Platform Graphics Rendering Engine
_________________________________________________________________


Introduction
Jamoma Graphics is an open source, cross-platform, C++ programming layer for drawing on various platforms and in various contexts.  Actual rendering is performed internal using Cairo, and the API is provided using the Jamoma Foundation runtime.


Features
- Rapid development
- Self-documenting and clear code-structure
- Cross-platform wrapper around platform-specific code for manipulating windows, etc.


Installing
Place the folder containing this ReadMe somewhere on your hard-drive. Then Make sure your compiler has access to it in order to compile the code.


Documentation / API
Doxygen generated documentation is located in the "documentation" folder.
